web programming . aj hansen

week one

Basic HTML

Set up a webpage that includes some biographical information to share with your peers.Only use your first name and you may change or delete this page later.

CSS Basics

Duplicate your first exercise and add internal styles to it. How might you call out certain content? Commit and sync to the repository.

Basic Layout

Start a new html file. Choose one of the Scandinavian flags and code it using divs, classes and floats. The size of the flag is up to you.

Positioning

Start a new html file and make a Scandinavian flag using divs, classes and absolute positioning.

Digital Bits

Make five drawings using div tags and css classes. Create an html file for each digital bit, and put them into a folder called "Digital Bits" in your github repo.

week two

CSS Grid

Make two flags using CSS grid. Use an implicit grid for the Nairu flag. Use an explicit one for the United States flag.

Grid Drawing

Assignment two asks that you play with the new css layout properties (grid and flexbox) to come up with abstract compositions. The webpages should change when the browser changes viewport widths.

week three

CSS Flexbox

Code up the following flags in separate html files using flexbox and external svgs provided. Commit and sync your work.

Media Queries

Using flex or grid, create a simple flag (your own country?). Change that flag to two other designs at two breakpoints (720px width and 360px width).

Interactive Pages

Code up the flag of Burkina Faso. On click, change it to the flag of Senegal.

Web Animation

Create a single animation that goes as far as you can go with motion. You do not need to use only abstract shapes; the look of the site is up to you.

week four

SVG Candy Cane

Copy the candy cane svg from the example file and paste into a new file. Color the candy cane the same as the candy cane you were given.

Personal Site

This final assignment asks you to create a single page website that collects your exercises and previous assignments. You should use CSS Grid and Flexbox for the overall structure of the webpage. Floats and positioning may also be needed.
